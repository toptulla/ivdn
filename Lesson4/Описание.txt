---------------- Селекторы ----------------

1. Сложные селекторы
    - Контекстные (h1 p) (1:34) - если необходимо определить стили для элемента вложенного в другой (родитель пробел элемент). Все элементы p внутри, не важно на каком уровне.
    - Соседние (h1 + p | h2 ~ p) (3:22) - если необходимо выбрать элементы, которые располагаются друг за другом.
    - Дочерние (h1 > p) (6:00) - дочерним называется элемент, который располагается НЕПОСРЕДСТВЕННО (в этом отличие от контекстного) внутри родительского элемента.
    - Селекторы атрибутов ()

Подключение пользовательских шрифтов, правило @font-face


selector1, selector2 {
    Стили для всех элементов selector1 и selector2 (стили группы).
}

selector1 selector2 {
    Применить стили ко всем вложенным в selector1 элементам selector2, не важно на каком уровне вложенности.
}

selector1 + selector2 {
    Применить стили к 1-у элементу selector2, который следует НЕПОСРЕДСТВЕННО за элеметом selector1.
}

selector1 ~ selector2 {
    Применить стили ко всем элементам selector2, которые следуют НЕПОСРЕДСТВЕННО за элеметом selector1 (на том же уровне).
}

selector1 > selector2 {
    Применить стили ко всем элементам selector2, которые являются прямыми потомками элемента selector1.
}
